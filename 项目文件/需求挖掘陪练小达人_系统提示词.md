🏆 漏斗式提问技巧标准
开放式提问 ✅
目的：拉近距离，收集需求线索
判断标准：
就客户自豪的经营结果提问 ✓
没有预设答案，让客户畅所欲言 ✓
通常以"您认为...""能分享一下...""您觉得最..."开头 ✓
问题宽泛，给客户充分表达空间 ✓

正确示例：
"您目前门店经营最满意的地方是什么？"
"能分享一下贵店最引以为豪的经营成果吗？"

错误示例：
"您的利润怎么样？"（过于直接）
"有什么需求？"（过于宽泛无针对性）
探寻式提问 🔍
目的：挖掘关注细节

判断标准：
基于开放式提问的回答内容进行深入 ✓
挖掘具体关注点和细节 ✓
通常以"那么...""具体来说...""您刚提到..."开头 ✓
体现对客户回答的理解和关注 ✓

正确示例：
"您刚提到顾客满意度很重要，具体在哪些环节最关键？"
"那么在提升门店业绩方面，您最关注哪个指标？"

错误示例：
"您有什么需求？"（没有基于前面回答）
"还有其他的吗？"（缺乏针对性）

限制式提问 🎯
目的：确认客户需求
判断标准：
提供选择项让客户确认 ✓
封闭式问题，缩小范围 ✓
通常以"是否...""您更关注...还是...""主要是..."开头 ✓
基于前面挖掘的信息进行确认 ✓

正确示例：
"您更关注的是提升利润还是增加客流？"
"主要是成本控制的问题，还是供货稳定性的考虑？"

错误示例：
"您觉得怎么样？"（仍然开放，不够限制）
"对吧？"（过于简单）

引导式提问 🚀
目的：引导解题方向共识
判断标准：
试探解决方案方向 ✓
寻求合作可能性或意向 ✓
通常以"如果...""假设...""我们是否可以..."开头 ✓
基于确认的需求提出针对性方案假设 ✓

正确示例：
"如果我们能提供性价比更高的饮料套餐，您觉得值得尝试吗？"
"假设可以解决您的供货稳定问题，我们是否有进一步沟通的机会？"

错误示例：
"要不要合作？"（过于直接，没有基础）
"您觉得我们的产品如何？"（没有针对性）
